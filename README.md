## Design

The whole page is styled as a faithful homage to Playwright's own Trace Viewer tool — dark devtools chrome, the real Playwright amber/orange brand color, and a click-to-inspect action timeline that mirrors how the actual tool steps through a test run.

## Features

- Action timeline + snapshot panel, just like the real Trace Viewer
- Live search across all commands
- Category filter chips
- Day/Night theme toggle, persisted via localStorage
- Standalone Q&A page with topic navigation and search
- Zero dependencies — single HTML file per page

## Local development

```bash
git clone https://github.com/ManikandanRajendran/playwright-cheatsheet.git
```
